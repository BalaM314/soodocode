# Soodocode

Soodocode: a runtime for Cambridge CS (9618) pseudocode syntax.

## Features
* [x] Basic statements
* [ ] FOR STEP statement
* [x] Basic operators
* [x] n-dimensional arrays
* [x] Functions/Procedures
* [x] CASE OF statements
* [x] Builtin functions
* [x] Enums
* [x] Pointers
* [x] Records
* [x] Shortened forms (for variable declaration and function arguments)
* [ ] Sets
* [ ] Files
* [ ] Classes
